# Telemost Converter

Набор PowerShell-скриптов для автоматической обработки записей видеовстреч из **Яндекс.Телемост**.

## Цель проекта

Создание **сортируемого архива записей**, подготовленных к **транскрибации**:

- Многие сервисы транскрибации не поддерживают формат `.webm`
- Конвертация в `.mp4` обеспечивает совместимость с любыми сервисами
- Извлечение аудио в `.mp3` с удалением тишины из аудио экономит минуты при оплате за время расшифровки и повышает качество транскрибации
- Автоматическое именование по дате упрощает поиск и организацию архива

## Возможности

- **Конвертация** `.webm` → `.mp4` (видео) + `.mp3` (аудио)
- **Переименование** файлов по дате и времени записи
- **Интерактивный режим** — ввод пользовательского описания для каждой встречи
- **Автоматическое удаление тишины** из аудио
- **Архивирование** оригиналов в отдельную папку

## Требования

| Компонент  | Версия | Примечание                  |
| ---------- | ------ | --------------------------- |
| PowerShell | 5.1+   | Встроен в Windows 10/11     |
| FFmpeg     | Любая  | Должен быть доступен в PATH |

### Установка FFmpeg

**Windows (Chocolatey):**

```powershell
choco install ffmpeg
```

**Windows (Winget):**

```powershell
winget install Gyan.FFmpeg
```

**Вручную:**

1. Скачайте сборку с [gyan.dev](https://www.gyan.dev/ffmpeg/builds/)
2. Распакуйте в `C:\ffmpeg`
3. Добавьте `C:\ffmpeg\bin` в переменную окружения `PATH`

## Скрипты

### 1. convert.ps1

Конвертирует записи `.webm` в `.mp4` (видео) и `.mp3` (аудио).

#### Параметры

| Параметр      | Тип    | Описание                         |
| ------------- | ------ | -------------------------------- |
| `-rewrite`    | switch | Пересоздать существующие файлы   |
| `-skipMp3`    | switch | Пропустить конвертацию в MP3     |
| `-skipRename` | switch | Пропустить переименование файлов |

#### Примеры использования

**Полная обработка (переименование + MP4 + MP3):**

```powershell
.\convert.ps1
```

**Только видео (без MP3):**

```powershell
.\convert.ps1 -skipMp3
```

**Без переименования:**

```powershell
.\convert.ps1 -skipRename
```

**Пересоздать существующие файлы:**

```powershell
.\convert.ps1 -rewrite
```

#### Процесс работы

1. Проверка наличия FFmpeg
2. Создание папок `out/` и `del/` (если нет)
3. Запуск `fixfilenamedate.ps1` (если не указан `-skipRename`)
4. Конвертация каждого `.webm`:
   - `.webm` → `out/*.mp4` (видео, H.265, AAC 96k)
   - `.webm` → `out/*.mp3` (аудио, 80k, удаление тишины)
5. Перемещение оригинала в `del/`

### 2. fixfilenamedate.ps1

Переименовывает файлы, извлекая дату и время из имени.

#### Формат исходного имени

```
Запись встречи 15.03.26 14-30-00 — описание.webm
                    │    │    │
                    │    │    └─ секунды
                    │    └────── минуты
                    └─────────── часы
```

#### Формат результата

```
YYYYMMDD_HHMMSS описание.webm
20260315_143000 описание.webm
```

#### Параметры

| Параметр       | Тип    | Описание                             |
| -------------- | ------ | ------------------------------------ |
| `-folderPath`  | string | Путь к папке (по умолчанию `.`)      |
| `-rewrite`     | switch | Перезаписать существующие файлы      |
| `-interactive` | switch | Запросить описание для каждого файла |

#### Примеры использования

**Автоматическое переименование:**

```powershell
.\fixfilenamedate.ps1
```

**С вводом описания для каждого файла:**

```powershell
.\fixfilenamedate.ps1 -interactive
```

**Обработка другой папки:**

```powershell
.\fixfilenamedate.ps1 -folderPath "C:\Recordings"
```

## Типовой сценарий использования

### Шаг 1. Подготовка

1. Поместите файлы `.webm` из Яндекс.Телемост в корень проекта
2. Откройте PowerShell в папке проекта

### Шаг 2. Запуск конвертации

```powershell
# Полная обработка с переименованием
.\convert.ps1

# Или только конвертация без переименования
.\convert.ps1 -skipRename
```

### Шаг 3. Результат

```
out/
├── 20260315_143000 совещание.mp4    # Видео
└── 20260315_143000 совещание.mp3    # Аудио

del/
└── Запись встречи 15.03.26 14-30-00 — совещание.webm  # Оригинал
```

## Настройки конвертации

### Видео (MP4)

| Параметр | Значение     |
| -------- | ------------ |
| Кодек    | H.265 (HEVC) |
| CRF      | 26           |
| Preset   | slow         |
| Аудио    | AAC 96 kbps  |

### Аудио (MP3)

| Параметр  | Значение                 |
| --------- | ------------------------ |
| Кодек     | MP3 (LAME)               |
| Битрейт   | 80 kbps                  |
| Частота   | 44100 Гц                 |
| Каналы    | 1 (моно)                 |
| Обработка | Удаление тишины (-25 dB) |
